# binance 前端界面


## Build Setup

!!! node 版本请保持在 v16, v18 构建会报错

```bash
# clone the project
git clone https://github.com/sorry510/grid_trade_front.git

# enter the project directory
cd grid_trade_front

推荐使用 pnpm 次之 yarn

pnpm

# develop
pnpm dev
```

## Build

```bash

# build for production environment
npm run build:prod
```
